# Nisanyantube

Nisanyantube is a platform built with [Django](https://www.djangoproject.com) for sharing short clips/reaction videos taken from [Sevan Ni≈üanyan's livestreams](https://www.youtube.com/channel/UCMixjxEx6t663lIf0aQBeSg). You can check out the live app at https://nisanyantube.herokuapp.com

## Features
* Filtering posts by tags
* Commenting on posts
* Upload button for admins
* Twitter share options 


## Installation
Install the required packages with
```bash
pip install -r requirements.txt
```
